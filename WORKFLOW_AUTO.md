# WORKFLOW_AUTO.md - 自動化工作流程

## GitHub 同步協議

**強制要求:** 所有討論內容必須自動同步到 GitHub 儲存庫

### 儲存庫資訊
- **URL:** https://github.com/oceanicdayi/openclaw_data
- **類型:** 公開儲存庫
- **用途:** 保存所有 OpenClaw 討論記錄

### 同步流程
1. **討論結束後** 自動執行 git 操作
2. **git add .** - 添加所有變更
3. **git commit -m "[日期時間] 討論記錄"**
4. **git push origin main** - 推送到 GitHub

### 身份配置要求
- 需要設定 Git 使用者名稱和電子郵件
- 需要有效的 GitHub 個人存取權杖 (PAT)
- 需要確認遠端儲存庫連線正常

### 故障排除
- 如果推送失敗，立即回報使用者
- 檢查網路連線和權限設定
- 確認 GitHub 權杖有效性

## 記憶體管理

### 每日記憶體檔案
- **路徑:** `memory/YYYY-MM-DD.md`
- **用途:** 記錄當天的重要討論和決策
- **更新頻率:** 每次重要討論後即時更新

### 長期記憶體
- **路徑:** `MEMORY.md`
- **用途:** 保存長期重要資訊和學習
- **更新頻率:** 定期整理和精煉

## 安全協議

- 不儲存敏感個人資訊
- 所有內容自動同步到 GitHub 進行備份
- 遵循 Dayi 老師的資料保存要求

# 評估工具與方法

## 學習成效評估

### 知識理解評估

**選擇題範例**
```
1. 下列哪種地震波傳播速度最快？
   A) P波
   B) S波
   C) 表面波
   D) 雷利波

2. 台灣位於哪個地震帶？
   A) 環太平洋地震帶
   B) 地中海-喜馬拉雅地震帶
   C) 大西洋中脊地震帶
   D) 東非裂谷地震帶
```

**簡答題範例**
```
1. 請說明P波和S波的主要差異（至少三點）
2. 解釋為什麼地震預警系統可以提供數秒到數十秒的預警時間
3. 說明「趴下、掩護、穩住」三個步驟的重要性
```

### 技能應用評估

**實作評量清單**
- [ ] 能正確演示地震避難姿勢
- [ ] 能操作簡易地震偵測裝置
- [ ] 能解讀地震警報訊息
- [ ] 能制定個人防災計畫
- [ ] 能協助他人進行地震避難

**專題評分標準**
```
專題：設計抗震建築模型

評分項目：
- 結構穩定性（40%）
- 創意設計（30%）
- 團隊合作（20%）
- 報告呈現（10%）
```

## 態度與價值觀評估

### 防災態度量表

**地震防災態度問卷**
（5點量表：1=非常不同意，5=非常同意）

1. 我認為地震防災準備很重要
2. 我會定期檢查家中的防災物品
3. 我知道地震發生時該如何保護自己
4. 我願意參與社區防災演練
5. 我會提醒家人注意地震安全

## 不同年齡層評估工具

### 幼兒園與低年級
- **圖像選擇**：選擇正確的避難動作圖片
- **行為觀察**：觀察遊戲中的安全行為
- **口頭問答**：簡單的是非問題

### 中年級
- **情境反應**：描述地震情境下的正確做法
- **實作評量**：完成簡單的防災任務
- **小組討論**：參與防災計畫討論

### 高年級與國中
- **知識測驗**：地震科學知識評估
- **專題製作**：防災相關專題作品
- **模擬演練**：地震情境應變能力

### 高中與大學
- **研究報告**：地震相關主題研究
- **數據分析**：地震數據解讀與分析
- **方案設計**：防災系統或政策設計

## 評估頻率與時機

### 課前評估
- 先備知識測驗
- 學習興趣調查
- 防災經驗分享

### 課中評估
- 即時回饋問答
- 小組活動觀察
- 實作過程記錄

### 課後評估
- 學習成果測驗
- 技能實作評量
- 態度行為問卷

## 評分標準範例

### 知識測驗評分
```
90-100分：精通所有地震學概念，能應用於新情境
80-89分：熟練核心概念，能正確解答問題
70-79分：理解基本概念，有些應用困難
60-69分：知道部分概念，需要更多練習
60分以下：需要重新學習基礎概念
```

### 實作評量評分
```
優秀：動作標準熟練，能指導他人
良好：動作正確，能獨立完成
普通：需要提示才能完成
待加強：無法正確執行動作
```

## 評估資料運用

### 教學調整
- 根據評估結果調整教學節奏
- 針對弱點設計補救教學
- 強化學生表現良好的部分

### 學生回饋
- 提供具體的改進建議
- 肯定學生的進步與努力
- 設定個人化的學習目標

### 課程改善
- 分析整體學習成效
- 調整教學活動設計
- 更新教學資源內容

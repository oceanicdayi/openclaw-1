# 教學活動設計

## 互動教學方法

### 1. 探究式學習
**地震成因探究**
- 活動：使用黏土模擬板塊運動
- 材料：不同顏色黏土、壓克力板
- 步驟：學生分組模擬板塊碰撞、分離、錯動
- 討論：觀察產生的「斷層」和「地震」

### 2. 實驗操作
**建築物抗震測試**
- 活動：建造不同結構的建築模型
- 材料：積木、紙張、膠帶、振動平台
- 測試：比較不同結構的抗震性能
- 分析：討論抗震設計原則

### 3. 模擬體驗
**地震預警體驗**
- 活動：模擬地震預警時間計算
- 工具：地圖、計算器、計時器
- 情境：給定震央位置，計算預警時間
- 應用：討論預警時間的實際用途

### 4. 角色扮演
**防災指揮中心**
- 活動：模擬地震應變指揮
- 角色：指揮官、通訊員、救援隊、醫療組
- 情境：發生大地震後的應變決策
- 目標：學習團隊合作與危機處理

## 分齡活動設計

### 小學（1-3年級）
- **地震感覺遊戲**：用搖晃墊體驗不同震度
- **安全姿勢練習**：趴下、掩護、穩住動作練習
- **防災包準備**：認識防災物品的重要性

### 小學（4-6年級）
- **簡易地震儀**：製作瓶裝地震偵測器
- **地震知識問答**：分組競賽學習地震知識
- **社區防災地圖**：繪製學校周邊安全地點

### 國中
- **板塊運動模型**：建立台灣板塊互動模型
- **建築結構實驗**：測試不同建築材料的抗震性
- **預警系統設計**：設計校園地震預警方案

### 高中
- **地震數據分析**：分析歷史地震序列
- **抗震工程設計**：設計抗震建築結構
- **災害應變計畫**：制定家庭防災計畫

### 大學
- **斷層力學計算**：計算斷層應力累積
- **波傳理論應用**：模擬地震波傳播
- **預警算法開發**：開發簡易預警算法

- 研究防災政策
